APP OVERVIEW

アプリ概要

受験生・学生向けの「強制力 × StudyWithMe × 勉強Vlog × AIコーチング」を組み合わせた勉強継続アプリ。目的は「勉強するしかない環境を作る」「努力をコンテンツ化する」「受験生活を作品化する」こと。

高校生 浪人生 資格勉強の大学生 StudyWithMe好き setlog・勉強Vlog好き 勉強が続かない人
① 強制力サボれない環境を作る
② コンテンツ化努力をVlog・記録として残す
③ AI伴走毎日AIが分析・フィードバック
④ 仲間全国の受験生と一緒に勉強

MVP

最初に作る機能（詳細）

最優先は「本当に勉強継続率が上がるか」の検証。まずは起床強制・カメラON勉強・AIフィードバックを中心に実装します。

起床強制

  • 起床時間設定、アラーム、AI音声、電話通知
  • 顔認証で解除
  • 起床後、机に向かうまで追跡
  • ゴールは「起きる」ではなく「勉強開始」

勉強開始強制

  • 勉強開始時間を設定し、開始時にカメラON必須
  • 手元撮影 + 顔 or 手元認識
  • カメラOFFで失敗扱い
  • アプリ離脱でタイマー停止 / 他アプリ移動検知
  • UIは監視感より Lo-fi・cafe風・StudyWithMe風の落ち着く設計

StudyWithMeモード

  • タイマー / ポモドーロ / 集中時間計測
  • 勉強中録画 / タイムラプス保存
  • Vlogモード と StudyWithMeモードを選択可能

AI勉強分析（毎晩）

  • 今日の集中分析 / 科目割合 / 朝勉成功率 / 集中切れ時間 / 改善点
  • AI人格選択: 鬼コーチ / 優しい先輩 / 東大生風 / 体育会系
  • 数字だけではなく感情のあるフィードバック

自動勉強Vlog生成

  • 1日の記録を自動編集（朝起床・通学・カフェ・勉強・夜勉・タイムラプス）
  • 勉強時間 / 今日の科目 / AIコメント / 日付を自動挿入
  • TikTok/Reelsサイズで30秒〜1分を出力
  • setlog風 / 浪人生Vlog風 / aesthetic study vlog

RETENTION DESIGN

継続設計（超重要）

人が続かない理由は「始められない」「孤独」「努力が見えない」「サボっても損がない」。これを設計で解決します。
